Detailed Description

This class implements a function that is 0 everywhere, except in a single point. DESCRIPTION 't' is the only abscissa (time) where the function is not 0. 'value' is the value of the function 't'. Both 't' and 'value' are pointers, rather than numbers, so that their state (initialized or not) can be checked.
