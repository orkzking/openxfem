Detailed Description

^ noded isoparametric triangle elements for plane elasticity

This class implements an isoparametric four-node quad plane- strain elasticity finite element. Each node has 2 degrees of freedom. The connectivity of element : [cornerNode_1 cornerNode_2 cornerNode_3 midNode_1 midNode_2 midNode_3]

DESCRIPTION : One single additional attribute is needed for Gauss integration purpose : 'jacobianMatrix'. This 2x2 matrix contains polynomials. TASKS :

  • calculating its Gauss points ;
  • calculating its B,D,N matrices and dV.
