Detailed Description

Enrichment detector, abstract base class.

An enrichment item based on the enrichment dectetor to enrich nodes. The kind of EnrichmentDetector will be chosen by read from the data file. It's necessary to make sure that the EnrichmentDetector is compatible with the EnrichmentItem.
