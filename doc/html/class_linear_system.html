Detailed Description

DESCRIPTION : This class defines the attribute 'leftHandSide', 'rightHandSide' and 'solutionArray'. Attribute 'currentSize' is used during the equation numbering process (see below). TASKS :

  • solving itself (implemented by the subclasses of LinearSystem) ;
  • assigning an equation number to any degree of freedom which asks for one (method 'giveUpdatedCurrentSize'). The system does so by returning its attribute 'currentSize' incremented every times by 1.
