Detailed Description

This abstract class is the superclass of the classes that implement functions y = f(t) , where t is the time. These function are used for weighing the loads (see TJR Hughes, "The Finite Element Method", p 677). A load-time function is attribute of the domain. It is usually also at- tribute of one or more loads. DESCRIPTION The parameters which describe the function are defined by the subclasses of this class. TASK Returning the value 'y' at any abscissa 't'.
