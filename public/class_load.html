Detailed Description

Loads : Boundary conditions, Nodal loads, Dead loads.

This abstract class is the superclass of the classes that implement loads (body load, nodal load, boundary conditions, etc). A load is an attribute of the domain. It is usually also attribute of several elements, nodes or dofs. DESCRIPTION The load stores its values in 'componentArray'. The components of a load at a given time step is the product of 'componentArray' by the value of the function 'loadTimeFunction' at that time step. TASK Returning its components and its load-time function ;
