Detailed Description

This class implements key/value associations, e.g., the key 'E' and its value 210000. A pair is used as an entry in a dictionary. DESCRIPTION : A pair has three components : its name (a character), its value (a number), a pointer to the next pair in the dictionary. TASKS :

  • Returning its key, or its value, or the next pair ;
  • Appending another pair to itself.
