Detailed Description

A general class for matrices.

This abstract class is the superclass of the class that implement matrices (FloatMatrix, PolynomialMatrix,...). DESCRIPTION : A matrix is characterized by its number of rows and columns. TASKS : Its tasks are defined in the subclasses.
