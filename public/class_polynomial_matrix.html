Detailed Description

This class implements a matrix which contains polynomials. These matrices are typically used as jacobian matrices of finite elements. DESCRIPTION : The matrix stores its nRows*nColumns polynomials column by column. TASKS :

  • storing and returing coefficients (i.e., polynomials), in method 'at' ;
  • evaluating itself at a given point (method 'EvaluatedAt'). This prod- uces a matrix of the same size, containing numbers.
