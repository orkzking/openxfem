Detailed Description

Material Interface.

This class implements a material interface. Each Material Interface is associated to two Materials, the part below this interface is assumed belonging to the ith material, and the upper part is of the (i+1)th material. The corresponding enrichment function is the absolute of signed distance, which is implemented in class AbsoluteSignedDistance. (

See also
paper "Arbitrary discontinuities in finite element" of Prof. TB) The input file is : 1 MaterialInterface Mat 2 1 2 geometry 1 enrFuncs 1 1 enrichScheme 3
