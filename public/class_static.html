Detailed Description

This class implements a quasi-static time integration scheme. Multiple time steps in quasi-statics are useful for computing a structure subjected to various load cases. The static scheme assigns to the dofs a unique unknown :

  • the displacement 'd' . The static scheme assumes that the linear system's left-hand side remains unmodified all along the time history.
