Detailed Description

A geometry entity is geometry of an enrichment item.

This class is an abstract class, the superclass of all classes that imple- ment different types of geometry entities: Vertex, PiecewiseLinear, PicewiseParabolic, Circle and Ellipse.
DESCRIPTION
Enrichment items will use this geometry to find enriched nodes.
